## Bug report
When submitting a bug report, please make sure your issue contains enough information for reproducing the problem. A video recording of the problem can be very helpful

## Feature request
If you have an idea for a new feature or improvement for Smoking Tracker, please provide enough information to help us understand and implement it

Include details such as
- What the feature is and how it should work
- Why it would be useful or improve the app
- Optional: mockups, sketches, or screenshots that illustrate your idea

## Pull request
If you want to contribute code directly to the project, please follow these steps

1. **Comment on an existing issue** or create a **new issue** describing the changes you plan to make
2. **Avoid duplicate work**: Let others know you are working on the issue by leaving a comment or requesting the issue to be assigned to you

Following these steps helps us coordinate contributions and ensures that multiple pull requests do not attempt to resolve the same problem
